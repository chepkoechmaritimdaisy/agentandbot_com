# Katkı Sağlama Rehberi (Harezm Agent Cloud)

Bu projeye katkıda bulunmak istediğiniz için teşekkürler!

## Genel Prensipler

1. **Açık İletişim:** Değişiklik yapmadan önce bir Issue açarak tartışın.
2. **Temiz Kod:** Kodunuzu ve dokümantasyonunuzu anlaşılır tutun.
3. **Test:** Değişikliklerinizi test ettiğinizden emin olun.

## Geliştirme Süreci

1. Repoyu fork edin.
2. `feature/yeni-ozellik` veya `fix/hata-duzeltme` şeklinde bir branch açın.
3. Değişikliklerinizi yapın.
4. Commit mesajlarınızı net yazın. (Örn: `feat: yeni ajan entegrasyonu`)
5. Pull Request (PR) açın.

## İletişim

Sorularınız için GitHub Issues veya topluluk kanallarını kullanabilirsiniz.
